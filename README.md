# Proyecto de Prácticas Git

## Descripción del proyecto

Este repositorio contiene una colección de ejercicios prácticos para aprender y utilizar Git y GitHub. Se trabajan conceptos como repositorios, commits, ramas, merges, etiquetas, conflictos y sincronización con repositorios remotos.

## Funcionalidades

* Creación y configuración de repositorios Git.
* Gestión de commits y control de versiones.
* Creación y administración de ramas.
* Resolución de conflictos de fusión.
* Uso de etiquetas (tags) y releases.
* Sincronización con GitHub.

## Comandos Git básicos

| Comando                    | Descripción                         |
| -------------------------- | ----------------------------------- |
| `git init`                 | Inicializa un repositorio Git       |
| `git status`               | Muestra el estado del repositorio   |
| `git add .`                | Añade cambios al área de staging    |
| `git commit -m "mensaje"`  | Crea un commit                      |
| `git log --oneline`        | Muestra el historial resumido       |
| `git branch`               | Lista las ramas                     |
| `git checkout nombre-rama` | Cambia de rama                      |
| `git merge nombre-rama`    | Fusiona una rama                    |
| `git push origin main`     | Envía cambios al repositorio remoto |

## Ejemplo de código

```bash
git add .
git commit -m "Actualización del README"
git push origin main
```
